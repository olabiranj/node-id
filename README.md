# node-id
A random unique id generator"

## Quick Start


```bash
# Install dependency
npm i node-id 

# Import dependency
const nodeid = require('node-id');

# Initialize variable
let uniqueId = nodeid();

# Use id
console.log(uniqueId);

# Manually input length of id(8-50)
let uniqueId = nodeid(30);

```
